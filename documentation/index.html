Changing slider images:

Naos use Flex Slider to display slider images. If you want to change them, simply upload your images in images/slider folder and link to them in index.html file. If you want to add more slides simply add more "li" tags. You can find more about Flex Slider customization on their site. Keep the same image width-height ratio in order to images display properly in the circle.

Testimonial images:

Testimonial images are located inside images folder. These images should also have the same with-height ratio in order to display properly.

Contact

Update contact form with your email address

Open send.php file and update "$adminemail" with your email address.

Google map:

Google map is iframed. If you want to change it go to Google maps, enter your location, click on link button and pick up the iframed version of the link. Update your google map by replacing only the src path inside index.html file. Leave id="map" inside iframe so map could be responsive.
